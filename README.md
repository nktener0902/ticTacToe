# ticTacToe
Tic Tac Toe app. This program is created for my Javascript skill up.

# How to play

* Install npm, nodejs.
* Move to this direcotry with a shell/command prompt.
* `set DEBUG=app:* & npm start`  
(windows)
* Open browser, and http://localhost:3000/
